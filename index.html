String Utils

This is a basic online string utility tool to facilitate string functions online. Json is a widely used notation and hence its pretty view, conversions to xml, php array and various other string tools are available on this site.

  • Includes conversion to uppercase, lowercase and camelcase.
  • Valid json structure to xml and php array.
  • Outputs are formatted to enable opening and closing of nodes.
  • Encoding / Decoding to base 64
  • Url encoding /decoding
